IP Geolocation Tracker (GUI Version)

A simple and interactive Python GUI tool that lets you track the geolocation of any IP address and view it on a world map using Folium.
It also logs results in a CSV file for easy record-keeping.

Features

✅ One-click IP lookup (GUI popup)
✅ Fetches city, region, country, ISP, timezone, and coordinates
✅ Automatically detects your public IP (if left blank)
✅ Generates an interactive map (ip_location.html)
✅ Logs lookups in ip_log.csv
✅ Exception logging in error_log.txt

 Tech Stack

Language: Python 3

Libraries: requests, folium, tkinter

APIs Used:

ipinfo.io

ip-api.com
